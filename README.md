# Math calculator

- 지원되는 기능
    - 수식 및 x, y 에 대입할 값 각각 입력
    - Canonical form 리턴
    - Derivative form 리턴
    - x, y 에 대입한 결과값 리턴
    - Continuity 여부 리턴
    - Differentiability 여부 리턴
    - Canonical 및 Derivative form 의 Plot 리턴
    - GUI 구현 (Gradio)
    
## Install
```
pip install gradio

pip install matplotlib
```


## Usage
```
python calculator.py
```
